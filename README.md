If you are into good practices add some theme-colored editor rulers to your vscode code as these ones:

```python
"editor.rulers": [
{
"column": 79,
"color": "#ff40806f"
},
{
"column": 83,
"color": "#ff4080a8"
},
{
"column": 87,
"color": "#ff4080e1"
}
]
```
